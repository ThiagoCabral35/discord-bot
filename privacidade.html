Política de Privacidade

1. Introdução: Esta Política de Privacidade descreve como as informações são tratadas pelo Biricutico no Discord, criado para uso recreativo entre amigos. O objetivo principal é fornecer entretenimento, sem coleta ou armazenamento de dados pessoais.

2. Dados Coletados:

  • O bot não armazena dados pessoais de seus usuários.
  • Todas as informações processadas são limitadas a:
    • Identificação de comandos enviados no servidor Discord.
    • Dados públicos disponíveis no Discord (como nomes de usuários e mensagens enviadas no chat).

3. Uso das Informações: As informações processadas pelo bot são usadas apenas para responder aos comandos e fornecer as funcionalidades do Biricutico. Nenhum dado é compartilhado com terceiros.

4. Segurança: O bot utiliza as práticas padrão para garantir a segurança das interações no servidor Discord. Como o bot não armazena informações, não há riscos associados a vazamentos de dados.

5. Direitos dos Usuários: Os usuários têm o direito de desativar o bot ou removê-lo de um servidor Discord a qualquer momento. Como não há coleta de informações pessoais, não há dados para serem excluídos ou alterados.

6. Alterações nesta Política: A Política de Privacidade pode ser atualizada a qualquer momento, sem aviso prévio. É recomendável revisar este documento periodicamente para se manter informado.
